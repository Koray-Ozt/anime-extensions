# Koray-Ozt Anime Extensions

Anikku ve Aniyomi için kişisel anime extension deposu.

## Kurulum

Uygulamanın extension repository alanına şu adresi ekleyin:

```text
https://raw.githubusercontent.com/Koray-Ozt/anime-repo/repo/index.min.json
```

İlk kaynak: **AnimeciX** — https://animecix.tv

APK ve katalog dosyaları [anime-repo](https://github.com/Koray-Ozt/anime-repo/tree/repo) reposunun
`repo` dalında yayımlanır.

Bu proje içerik sağlayıcılarla veya Anikku/Aniyomi geliştiricileriyle bağlantılı değildir.
Kaynak altyapısı [yuzono/anime-extensions](https://github.com/yuzono/anime-extensions) projesinden
çatallanmıştır ve kendi lisansı altında kullanılır.
